# Jayne's Portfolio

Personal portfolio/resume website showcasing customer success leadership background. Animated, modern design.

## Tech Stack
- React 18, TypeScript, Vite
- TailwindCSS
- Framer Motion (animations)
- lucide-react (icons)

## Key Details
- Resume is embedded in the site
- Focus on customer success / leadership background
- Keep animations smooth and professional

## Conventions
- TypeScript throughout
- Framer Motion for all animations
- TailwindCSS for styling
